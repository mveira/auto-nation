# Vehicle Images Directory

## Place your vehicle photos here

This directory should contain all the actual photos of your vehicles. 

### Required Images

See the main project file `VEHICLE-IMAGES-SETUP.md` for the complete list of required images.

### Quick Reference

Each vehicle needs at least 2 images:
- `[vehicle-name]-1.jpg` - Main featured image
- `[vehicle-name]-2.jpg` - Gallery/interior image

### Example

For a BMW M2:
- `bmw-m2-1.jpg`
- `bmw-m2-2.jpg`
- `bmw-m2-3.jpg` (optional)
- `bmw-m2-4.jpg` (optional)

### Image Specifications

- **Format**: JPG
- **Size**: 1200x800px (landscape)
- **File size**: 200-300KB (compressed)
- **Quality**: 80-85% JPEG quality

### Your CSV indicated you have photos!

According to your exported forecourt CSV, you have between 2-42 photos per vehicle. Upload those photos here using the naming convention specified in `VEHICLE-IMAGES-SETUP.md`.
